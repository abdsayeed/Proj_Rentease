# Rentease Frontend

Modern Angular application for the Rentease property rental platform.

## Features Implemented

✅ **Home Page** - Property listings with search and filters
✅ **Login** - User authentication
✅ **Register** - New user registration with role selection
✅ **Property Details** - Detailed property view with inquiry form
✅ **API Service** - Complete backend integration
✅ **Responsive Design** - Works on all devices

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
ng serve
```

3. Open browser at `http://localhost:4200`

## Backend Setup

Make sure Flask backend is running:
```bash
cd ..
pip install -r requirements.txt
python run.py
```

Backend runs on `http://127.0.0.1:5000`
